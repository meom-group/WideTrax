## Installation

`widetrax` is Pip-installable:

.. code-block:: python
   pip install widetrax


Or, if you prefer, you can install it directly from the GitHub repository:

.. code-block:: python
   git clone https://github.com/meom-group/widetrax.git
   cd widetrax
   pip install .
